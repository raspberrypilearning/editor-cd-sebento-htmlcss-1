<h2 class="c-project-heading--task">Style the heading</h2>

For `<h1>` headings, you use the `h1` selector.

--- task ---

Add a CSS rule for `h1`.

--- /task ---

<div class="c-project-code">
--- code ---
---
language: css
filename: styles.css
line_numbers: true
line_number_start: 1
line_highlights: 7-10
---
body {
  background-color: LightSkyBlue;
  font-family: "Helvetica", sans-serif;
  color: purple;
}

h1 {
  color: orange;
  font-family: "Times New Roman", serif;
}

--- /code ---
</div>

--- task ---

**Run** your code.

- Your heading text should be orange now, with the paragraph in purple as before. 

--- /task ---
